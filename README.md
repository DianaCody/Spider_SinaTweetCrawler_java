
## 新浪微博爬虫（java版）

####*新浪微博爬虫*
#####实现自定义输入关键词，指定要爬取的页面数据，根据关键词提取页面中的微博信息数据。

> * 自定义搜索关键字
> * 自动获取代理IP爬取
> * 非登录，爬取当天微博信息数据存储于本地html文件
